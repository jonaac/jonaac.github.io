WORKS

Thesis - LTL Reward Specification in a CMDP

In Progress: Improving the safety and learning performance of Reinforcement Learning agents with reward specification via Linear Temporal Logic in Constrained Markov Decission Processes.

RL & Formal Logic languages: Literary Review

A literary review on the current research, scholar articles and resources that cover the conjoint topic of Reinforcement learning and Logic. The goal is to contextualize the existing literature on this topic, critically evaluate the literature and identify any potential gaps within the context of the existing literature.

Deep XGBoost Image Classifier

This project examines the possibility of combining CNN with eXtreme Gradient Boosting (XGBoost). The suggested model would integrate a CNN as a feature extractor with XGBoost as a recognizer to classify images based on the features outputted from the CNN. The new algorithm was evaluated with the CIFAR-10 databases using three different CNN structures, a baseline CNN, VGG16 and ResNet50.

Bitcoin price prediction with Deep RL

The goal of the project was to test and evaluate the viability of Deep Reinforcement Learning algorithms for Bitcoin price prediction.

Safety for Intelligent Agents

Developed a reinforcement Learning grid-world simulation from scratch (no ML or RL libraries) to experiment and test safety methods for intelligent agents. A planetary robot that moves around in a grid world collecting mineral ore and has to avoid falling into craters.
